# Full-Stack Movie Rental Web Application

## Overview
This project is a full-stack web application built for a fictional client, Internet Movies Rental Company (IMR).  
The application provides an internal portal for managing a movie database, including creating, viewing, editing, and maintaining movie records.

This repository demonstrates practical full-stack development skills, including frontend interface design, backend integration, and database-driven workflows.

## Architecture Summary
The application follows a standard full-stack structure:
- Frontend: Handles user interaction and form submission
- Backend: Processes requests and business logic
- Data layer: Stores and retrieves movie records

This separation supports easier debugging, scaling, and future feature expansion.
